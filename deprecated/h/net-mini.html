How Much RAM Do You Need?

Below are common activities per amount of RAM in a Net Mini.

2GB

Extremely light use

Offline tasks like programming

4GB

Casual use

Web browsing, programming online or offline

8GB

Regular use

Web browsing, programming, web games

NOTE: None of the Net Mini models are intended for intense gaming/graphics
